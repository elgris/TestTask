Тестовое задание для программиста.

Необходимо написать программу, которая будет отображать графики
функций выбранной из списка, определенного на этапе программирования.
Каждая функция может иметь от одного до трех параметров, вводимых
пользователем. Пользователь так же определяет интервал, на котором
будут производится построение графика и шаг, с которым будут
выбираться узловые точки.

Для написания программы можно использовать компоненты стандартной
библиотеки C++ и компоненты библиотеки Qt версии 4.6.0 или выше.

Использование сторонних библиотек и компонентов _не_ допускается.

Система сборки: qmake.

Программа должна без проблем собиратся и работать в среде Window7
с установленным QtSDK и в среде Ubuntu 10.04 с установленными
библиотеками Qt из основного репозитория Ubuntu.

Дополнительные файлы:
        1.png - интерфейс программы в минимизированном виде,
        1-number.png - интерфейс программы в минимизированном виде с пояснениями,
        2.png - интерфейс программы, развернутый на весь экран,
        3.png - интерфейс таблицы значений
        functions.txt - список функций.

Работа программы (числа в скобках поясняются в 1-number.png):

Пользователь выбирает одну из функций в списке (1),
после этого указывает параметры A, B, C, границы интервала и шаг
в (2).

Далее, по нажатию кнопки "Start" вычисляются значения функции в точках
интервала в соответствии с шагом. При этом по мере вычисления,
результаты вычисления отображаются на графике. Надпись на кнопке меняется
на "Progress (X%)" и отображает текущий прогресс вычислений. При этом
возможность ввода значений в (2) и выбора функции в (1) блокируется.

По нажатию кнопки "Pause" вычисления прекращаются, а надпись на кнопке
меняется на "Continue". По нажатию на кнопку "Continue" вычисления
продолжаются с того же места, где были прерванны, а надпись на кнопке
меняется на "Pause".

По нажатию на кнопку "Break" вычисления прекращаются, уже вычисленные
данные остаются доступными для пользователя, а кнопка "Progress (X%)"
меняет текст на "New".

При клике по кнопке "New" несохраненные данные уничтожаются, поле с графиком
очищается. Пользователю становятся доступны для ввода поля (2) и список (1).

Если программа завершила вычисления, либо если программа поставленна
на паузу, у пользователя есть возможность посмотреть текущие вычисленные
значения, кликнув по графику (4). В этом случае открывается окно с таблицей
значений.

При нажатии на кнопку "Save" программа позволяет пользователю сохранить
текущий прогресс вычислений в отдельный файл.

При нажатии на кнопку "Load" программа позволяет востановить сохраненное
состояние из файла, указанного пользователем (вычисленные данные, выбранная
функция, параметры функции и интервала) и может продолжить вычисления, если
они не были завершены.
